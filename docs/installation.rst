============
Installation
============


Stable release
--------------

To install wordle-clone, run this command in your terminal:

.. code-block:: console

    $ pip install wordle_clone

This is the preferred method to install wordle-clone, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for wordle-clone can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Nathan-Williams-1/wordle_clone

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/Nathan-Williams-1/wordle_clone/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Nathan-Williams-1/wordle_clone
.. _tarball: https://github.com/Nathan-Williams-1/wordle_clone/tarball/master
